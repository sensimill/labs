1. Для чого потрібен тег < div >?

використовується для завантаження і відображення об'єктів;
задає вирівнювання вмісту тега;
призначений для виділення фрагмента документа з метою зміни виду вмісту.

3. Для чого використовується атрибут "class"?

Приховує вміст елемента від перегляду;
Встановлює контекстне меню для елемента;
Визначає ім'я класу, яке дозволяє зв'язати тег зі стильовим оформленням.
